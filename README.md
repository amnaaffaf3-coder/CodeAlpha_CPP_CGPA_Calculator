# CodeAlpha CPP CGPA Calculator

A beginner-friendly C++ console application that calculates a student's final CGPA based on the grades and credit hours of multiple courses. This project was developed as part of the **CodeAlpha C++ Programming Internship**.

## Features

- Calculates CGPA for any number of courses
- Accepts course name, grade, and credit hours as input
- Converts letter grades into grade points
- Calculates the weighted CGPA accurately
- Simple and user-friendly console interface
- Uses loops, conditional statements, functions, and variables

## Technologies Used

- C++
- Standard Library (`iostream`, `string`)

## Project Files

- `CGPA_Calculator.cpp` – Source code
- `CGPA_Calculator.pdf` – Project documentation
- `README.md` – Project overview

## How to Run

1. Compile the `CGPA_Calculator.cpp` file using any C++ compiler.
2. Run the generated executable.
3. Enter the number of courses.
4. Enter each course's:
   - Course Name
   - Credit Hours
   - Letter Grade
5. The program will calculate and display the final CGPA.
